Темы, необходимые для проекта.
1.	Файловый ввод-вывод и работа с файловой системой
2.	Работа со временем
3.	Работа со строками и форматирование
4.	Многопоточность и управление процессами и задержками
5.	Парсинг аргументов командной строки
6.	Генерация псевдослучайных чисел
   
Эскиз

Эмулятор представляет собой консольное приложение. При запуске пользователь указывает коэффициент ускорения работы станка, скорость и количество строк-событий, способ ротации и кодировку файлов, которые также могут быть заданы по умолчанию. Программа работает в бесконечном цикле или до прерывания пользователе. Она генерирует события двух типов: штатные и ошибки. События имеют разную частоту, можно сделать соотношение частот событий также задаваемым или оставить конкретный по умолчанию. Программа будет создавать файлы в выделенной папке, путь к которой будет указан по завершении программы. При достижении размера любого из файлов в 5 мб происходит ротация. 

Примерная архитектура

1.	Модуль ядра
   
  a.	Парсинг командной строки
  
  b.	Инициализация остальных модулей
  
  c.	Запуск параллельных потоков записи
  
  d.	Ожидание завершения потоков и корректная обработка прерывания
  
3.	Модуль конфигурации
   
  a.	Определение всех основных констант (размер файлов, коэффициент ускорение, соотношение частот, пути к файлам и т.д.)
  
5.	Модуль генератора событий
   
  a.	Создание случайного статуса станка
  
  b.	Формирование строки, идущей на запись в файл
  
7.	Модуль времени
   
  a.	Получение и форматирование времени
  
  b.	Вычисление времени сна программы
  
9.	Модуль файловой системы
    
  a.	Открытие файлов и запись строк
  
  b.	Обновление файлов
